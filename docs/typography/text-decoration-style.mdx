Apply underlines, overlines, or strikethroughs.
>★ represents it's an iOS-only prop

## Import

```js title="src/screens/Component.js"
import { decoration } from "nativeflowcss";
```

## Properties

| Classes             | Native Props                        | Pattern Box                                                                                                                             |
| ------------------- | ----------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| `decoration.solid`  | `{ textDecorationStyle: "solid" }`  | <p style={{ width: "100px", height: "20px", textDecorationLine: "underline", textDecorationStyle: "solid" }}>solid</p>   |
| `decoration.dotted` | `{ textDecorationStyle: "dotted" }` | <p style={{ width: "100px", height: "20px", textDecorationLine: "underline", textDecorationStyle: "dotted" }}>dotted</p> |
| `decoration.dashed` | `{ textDecorationStyle: "dashed" }` | <p style={{ width: "100px", height: "20px", textDecorationLine: "underline", textDecorationStyle: "dashed" }}>dashed</p> |
| `decoration.double` | `{ textDecorationStyle: "double" }` | <p style={{ width: "100px", height: "20px", textDecorationLine: "underline", textDecorationStyle: "double" }}>double</p> |
